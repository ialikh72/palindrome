



# Palindrome Number Program in C#

## Introduction
This is a simple C# console application that checks whether a number is a palindrome or not.

A palindrome number is a number that reads the same forward and backward.

Examples:
- 121
- 1331
- 777

## Features
- Takes input from the user
- Reverses the number
- Compares original and reversed numbers
- Displays whether the number is palindrome or not

## Technologies Used
- C#
- .NET Console Application

## How the Program Works
1. User enters a number.
2. The program reverses the number using a loop.
3. The reversed number is compared with the original number.
4. If both are equal, the number is palindrome.

## Code Logic
The program uses:
- while loop
- modulus operator (%)
- division operator (/)

### Formula Used
Reverse Process:
reverse = (reverse * 10) + remainder
